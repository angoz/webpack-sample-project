# webpack-sample-project
webpack sample project

## Environment
> 1. node --version v6.11.0
> 2. npm --version 3.10.10
> 3. webpack --version 3.2.0

## Usage
> 1. git clone https://github.com/angoz/webpack-sample-project.git
> 2. npm install
> 3. npm start
> 4. http://127.0.0.1:8080
